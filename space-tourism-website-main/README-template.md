## Visão geral

### O desafio

Os usuários devem ser capazes de:

- Visualizar o layout ideal para cada página do site de acordo com o tamanho da tela do dispositivo
- Ver os estados de hover em todos os elementos interativos da página
- Navegar entre as páginas e alternar entre as abas para visualizar novas informações
